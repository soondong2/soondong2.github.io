## 프로그래머스 연속된 수의 합
- 출처 : https://school.programmers.co.kr/learn/courses/30/lessons/12903
- 난이도 : Level1

## 풀이
```python
def solution(s):
    answer = ''
    if len(s) % 2 == 1:
        mid = len(s) // 2
        answer = s[mid]
    else:
        mid = len(s) // 2
        temp = s[mid-1], s[mid]
        answer = ''.join(temp)
        
    return answer
```
