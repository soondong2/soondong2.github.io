## 프로그래머스 연속된 수의 합
- 출처 : https://school.programmers.co.kr/learn/courses/30/lessons/12910
- 난이도 : Level1

## 풀이
```python
def solution(arr, divisor):
    answer = sorted([a for a in arr if a % divisor == 0])
    
    if len(answer) == 0:
        answer = [-1]
        
    return answer
```
